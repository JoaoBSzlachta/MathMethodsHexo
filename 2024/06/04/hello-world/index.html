Math Methods Intro
2024-06-04 ｜

Olá, olá. Este blog foi construído para realização de uma atividade da unidade curricular de Cálculo Numérico: Métodos para Ajuste de Curvas, do curso de Ciência da Computação da UniFil.
Utilizando do hexo e do layout ZenMind, foi criado esse blog para ser um portifólio de outras atividades práticas realizadas em aulas. Atividades essas que consistiam em escrever métodos de ajuste de curva em código para que atráves de determinados inputs, um programa pudesse realizar os cálculos e devolver o polinômio referente ao dado conjunto de pontos.

Os códigos

Foram feitas 2 páginas estáticas que contiam 3 métodos: 2 de interpolação e 1 de regressão linear.
Os métodos de interpolação foram os de Newton (Diferenças Divididas) e de Lagrange. Já para regressão linear, foi utilizado o método dos Mínimos Quadrados.

Tecnologias

A principio, apenas para fim de estudo do funcionamento dos métodos, os códigos foram escritos em Python na plataforma Google Colab.
Uma vez validados e prontos, os códigos foram convertidos para JavaScript e foram aplicados em páginas web construidas com HTML e CSS.
O deploy das páginas ocorreu através do GitHub Pages e os códigos estão disponíveis para uso no GitHub.
